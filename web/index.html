Portfolio Project

Multi-Agent Research Pipeline

A production-style AI system that decomposes a topic, gathers evidence, retrieves local knowledge via RAG, and generates a markdown report.

Planner Agent

Breaks a topic into focused sub-questions.

Search Agent

Retrieves external evidence per sub-question.

RAG Agent

Queries ChromaDB for local reference context.

Writer Agent

Synthesizes findings into a downloadable markdown report.
